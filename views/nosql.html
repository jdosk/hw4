What is a NoSQL database?

A NoSQL, or non SQL, database is a database that isnt relational. This shift in style allows for some distinct advantages and disadvanges from relational databases. Since information in a NoSQL database doesn't necessarily need to be linked, other methods of storing and organizing data can be used. This allows data to be stored on multiple servers, allowing NoSQL databases to have excellent scaling capabilities. Additionally, having data replicated in multiple servers allows NoSQL databases to have very high availability and reduction in failure points.

Types of NoSQL databases:

  • Key Value
  • Document
  • Column
  • Graph
